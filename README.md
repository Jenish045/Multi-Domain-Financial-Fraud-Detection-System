# Multi-Domain Financial Fraud Detection System

A complete, fully runnable fraud detection system across 3 domains:
- **Credit Card Fraud** — Autoencoder (TensorFlow/Keras, unsupervised)
- **Insurance Claim Fraud** — Random Forest + XGBoost (Scikit-learn)
- **E-Commerce Order Fraud** — LSTM (TensorFlow/Keras, supervised)

Includes a unified Ensemble Voting Engine combining all 3 models and a real-time Streamlit dashboard.
